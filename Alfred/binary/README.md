## Alfred插件10进制转为2进制
=========
编码中经常遇到一个问题：
拿到一串long型数字，需要看到哪几位是1，这个插件就是来处理这样的问题的。

输入

```
bi 10
```
输出

```
其中第3 1位为1
10转换为二进制数为:1010
```
并且

- 通知中心记录
- 黏贴到剪贴板
